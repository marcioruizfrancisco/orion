Convenção Oficial de Commits

| Prefixo     | Utilização                   |
| ----------- | ---------------------------- |
| `docs:`     | Documentação                 |
| `feat:`     | Nova funcionalidade          |
| `fix:`      | Correção de bugs             |
| `refactor:` | Refatoração                  |
| `build:`    | Docker, dependências e build |
| `test:`     | Testes                       |
| `chore:`    | Manutenção geral             |
| `ci:`       | Integração contínua          |
